🌎 NotebookLM Bridge

Integre o NotebookLM com o Claude Code via MCP

📚 Abrir NotebookLM 📄 Ver no GitHub

⚡ Como usar no Claude Code

  1. Clone o repositorio:
    git clone https://github.com/paulacunhaadv7-cmd/notebooklm-claude-bridge
  2. Instale as dependencias:
    cd notebooklm-claude-bridge && npm install
  3. Configure o MCP no Claude Code adicionando ao seu claude_desktop_config.json
  4. Acesse pelo GitHub Pages:
    https://paulacunhaadv7-cmd.github.io/notebooklm-claude-bridge
